A simple JavaScript project that allows users to switch between **Light**, **Dark**, and **Colorful** themes.  
The selected theme is saved using **localStorage**, so it persists even after page refresh.

## Features
- Switch between Light, Dark, and Colorful themes
- Uses **JavaScript DOM manipulation** for dynamic theme changes
- **localStorage support** to remember user preferences
- Smooth background transitions
- Clean and simple UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open `index.html` in your browser
2. Click on the theme buttons to switch themes
3. Your selected theme will be saved automatically and applied on next visit
